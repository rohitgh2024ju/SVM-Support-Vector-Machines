### Support Vector Machines

1. Overview

SVM is a supervised learning learning algorithm that focuses on classification and regression. The goal is to find a decision boundary that separates different classes while maximizing the margin between them.

2. Problem Statement

Given a dataset

$$
D = {(x_i, y_i)}_{i=1}^n
$$
